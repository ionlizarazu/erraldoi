# Erraldoiak

**Erraldoiak** Euskal Herriko erraldoi eta konpartsen ondarea biltzen, dokumentatzen eta zabaltzen duen web plataforma interaktiboa da. Proiektu honen helburua gure kulturaren parte diren figura hauen informazio teknikoa eta grafikoa modu dotore eta irisgarrian eskaintzea da.

## Ezaugarri Nagusiak

- **Datu Dinamikoak**: Informazio guztia **Wikidatatik** kargatzen da denbora errealean (SPARQL bidez).
- **Bilatzaile Integratua**: Konpartsak eta erraldoiak izenaren, herriaren edo kideen arabera iragazteko aukera, testu nabarmentzearekin.
- **Ondarearen Mapa**: Erraldoi eta konpartsa guztien kokapen geografikoa mapa interaktibo baten bidez arakatu daiteke.
- **Fitxa Teknikoak**: Erraldoi bakoitzaren altuera, pisua, egilea, sorkuntza urtea eta irudikatzen duen pertsonaiaren informazio zehatza.
- **Wikimedia Integrazioa**:
  - Argazkiak **Wikimedia Commons**etik kargatzen dira.
  - Lotura zuzenak **Wikipedia** eta **Wikidata** orrietara.

## Teknologia

- **React 19**: Erabiltzaile interfaze moderno eta azkarra.
- **TypeScript**: Kode sendoa eta akatsik gabea bermatzeko.
- **Tailwind CSS v4**: Diseinu editoriala eta asimetrikoa ("The Living Archive" estetika).
- **React Router**: Orrialdeen arteko nabigazio arina.
- **Leaflet**: Mapa interaktiboen kudeaketa.
- **Vite**: Garapen ingurune eta eraikitzaile optimizatua.

## Garapena

Proiektua lokalean martxan jartzeko:

1. Instalatu dependentziak:

   ```bash
   npm install
   ```

2. Garapen zerbitzaria piztu:

   ```bash
   npm run dev
   ```

3. Build produkziorako:
   ```bash
   npm run build
   ```

## 📚 Iturriak

Proiektu honek Wikimedia mugimenduaren balioak babesten ditu. Informazio guztia komunitateak egindako lanari esker dago eskuragarri Wikipedia, Wikidata eta Wikimedia Commons proiektuetan.

## ✍️ Egiletza eta Garapena

Aplikazio hau **opencode** (AI ingeniari autonomoa) bidez garatu da, Euskal Herriko kultura eta ondarea digitalki zabaltzeko asmoz. Garapen prozesu osoan zehar, adimen artifizialak kodearen arkitektura, diseinu editoriala eta Wikidatarekiko integrazio dinamikoa prestatu ditu, erabiltzailearen gidaritzapean.
